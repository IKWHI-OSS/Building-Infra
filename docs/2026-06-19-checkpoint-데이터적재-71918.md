## 할 일 (순서)
1. 71918 **활용신청 승인** 확인.
2. `aihubshell -mode l -datasetkey 71918` 로 파일트리·filekey 확인(키 불필요).
3. `aihubshell -mode d -datasetkey 71918 -aihubapikey "$KEY" -filekey all` — **13GB라 단일 다운로드**(71388식 배치/빈패킹 불필요).
4. 무결성: `unzip -t` 통과 확인(0바이트/손상 가드).
5. GCS 업로드 → **size-match 검증** → 검증 통과 후에만 로컬 삭제. 71918 전용 prefix 권장(예: `${BK}/aihub-71918/`).
6. 적재현황은 상태표·`gsutil ls/du`로 판정. (`-mode l` 반복 조회 금지 — 비용)
